# Screenshots

Drop your captures here so the main README renders them. Recommended shots:

- **`demo.gif`** — a 10–20s screen recording: upload a scanned PDF → progress bar
  runs → select text → search + jump → draw an annotation → zoom (annotation
  stays). This is the single highest-impact asset for the repo.
- **`viewer.png`** — a page open with the `⚡ Digital text / 🔍 OCR` badge visible.
- **`search.png`** — a search with a highlighted match on the page.

Tips: record at a modest window size, keep it under ~5 MB, and use a real,
recognizable document (not lorem ipsum) so it reads as genuinely useful.
